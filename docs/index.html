Graphene Data Grid takes gform to the next level, providing a full tabular interface for sorting, searching, paginating, editing, updating, adding, and deleting data.

Just define your gform form, map your CRUD functions, and your'e ready to go!

Basic Usage


          bt = new GrapheneDataGrid({container:'#myGrapheneDataGrid',
            search: false,columns: false,upload:false,download:false,title:'Demo Content',
            edit:true,
            add: true,
            click: function(model){model.toggle();},
            delete:true,
            entries:[],
            count:4,
            schema:[
                {type:"text", name:"name", label:"Name", required:true},
            ], data: [
                {name:'fifth'},
                {name:'fourth'},
                {name:'third'},
                {name:'second'},
                {name:'first'},
            ]
          })
